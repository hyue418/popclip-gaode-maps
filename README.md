### 支持选中文本后直接跳转至高德地图
```yaml
#popclip
name: 高德地图
icon: iconify:solar:map-arrow-square-bold
url: https://ditu.amap.com/search?query=***
```
**安装方式：选中上方整个文本块，点击PopClip工具栏中的“安装扩展”即可。**
